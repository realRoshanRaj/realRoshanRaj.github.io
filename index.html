Path GENERATOR

Java-fx application
  View Code on Github

As part of the Frog Force Programming team, I designed an application to help generate and transfer pre-determined autonomous paths onto the robot. It includes real-time visualization with the click of a mouse as well as quick generation by running a Quintic Spline fitting algorithm. Once a generated path is saved into CSV format, it is exported to the robot efficiently (cutting down the previous time with FileZilla by almost 50%).
